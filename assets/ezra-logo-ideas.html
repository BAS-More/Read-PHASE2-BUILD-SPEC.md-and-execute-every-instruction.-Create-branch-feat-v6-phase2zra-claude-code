EZRA Logo Directions

Three directions that fit Ezra HaSofer and the governance software better. The live logo has already had the top badge removed.

Scribe + Code

Keeps Ezra unmistakably as a scribe while the bracket frame signals software immediately. This is the strongest balance of identity and product.

assets/ezra-logo-concept-scribe-code.svg

Keeper of Standards

Pushes the governance side harder and feels more authoritative. Best if EZRA should feel like the steward of rules, order, and architecture.

assets/ezra-logo-concept-standards-tablets.svg

Decision Network

Leans more technical and product-like. The connected nodes suggest ADRs, governance graphs, and multi-agent orchestration.

assets/ezra-logo-concept-decision-nodes.svg
Recommendation: Scribe + Code is the best fit for the current product.
